# 图标使用说明

要构建 Windows 可执行程序，你需要准备应用图标。

## 图标要求

### 格式
- **文件类型**: .ico
- **尺寸**: 256x256 像素（推荐，包含多个尺寸：16x16, 32x32, 48x48, 256x256）
- **颜色**: 支持透明通道（RGBA）

### 图标命名
- 主图标文件命名为 `icon.ico`
- 放置在 `assets/` 目录下

## 如何创建图标

### 方法1：在线转换工具
1. 准备一张 256x256 的 PNG 图片
2. 访问 [Icon Converter](https://www.icoconverter.com/)
3. 上传 PNG 图片
4. 下载生成的 .ico 文件
5. 重命名为 `icon.ico`
6. 放到 `assets/` 目录

### 方法2：使用专业软件
- **Adobe Photoshop** + ICO 插件
- **GIMP** (免费) - 支持直接导出 ICO
- **IcoFX** (专业图标编辑器)

### 方法3：使用 Node.js 工具
```bash
npm install -g convert-icon
convert-icon input.png output.ico --sizes 16,32,48,256
```

## 测试图标

```bash
# 创建完成并测试运行
npm run build:dir
cd dist/win-unpacked
.
班级任务统计助手.exe
```

## 如果没有图标

如果没有图标文件，可以临时删除 package.json 中的图标配置：

```json
"win": {
  // "icon": "assets/icon.ico",  // 注释或删除这一行
  "target": [
    ...
  ]
}
```

应用会使用默认的 Electron 图标。

## 图标设计建议

- 使用简洁的图形
- 考虑使用教育相关元素（书本、笔、泡泡等）
- 使用鲜明的颜色，便于识别
- 在小尺寸下仍然清晰可辨

```
assets/
├── icon.ico          # 主图标（必需）
└── 图标说明.md       # 本文件
```
